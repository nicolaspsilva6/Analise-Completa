Neste projeto, realizamos uma análise completa de um banco de dados, explorando sua estrutura, identificando as principais tabelas, relacionamentos e tipos de dados utilizados. Foram aplicadas técnicas de modelagem, normalização e análise de dados para extrair informações relevantes. Além disso, avaliamos a integridade dos dados, possíveis inconsistências e propomos melhorias para otimizar o desempenho e a organização do banco.
